Looking for free Gems in 99 Nights in the Forest? You're in the right place! We've got all the active codes that still work in December 2025, plus a quick guide on how to redeem them.

Active Codes (December 2025)

  • afterparty - 15 Gems
  • yay fishing - 2 Gems

This code works differently! Instead of using the normal code menu, you need to type "yay fishing" in the game chat while you're fishing. Here's how:

  1. Upgrade your campfire to Level 2
  2. Find the Fishing Shack
  3. Grab the Old Rod inside
  4. Start fishing at a fishing hole
  5. Type "yay fishing" in chat
  6. Collect your 2 Gems!

Note: The fishing code only works once per account.

How to Redeem Codes

Redeeming codes in 99 Nights in the Forest is super easy:

  1. Launch the game on Roblox
  2. Click the Diamond icon in the bottom left corner
  3. Click the "Codes" button on the right side of the menu
  4. Type or paste a code from the list above
  5. Click Submit to claim your free Gems!
